# Tic Tac Toe – Python / Pygame

Cette application propose un jeu de Tic Tac Toe (morpion) avec interface graphique, développé en **Python 3**, **Pygame** et **pygame_gui**.
Elle permet de jouer contre un autre joueur ou contre l’ordinateur, avec affichage graphique des cases, symboles X et O, et écran de victoire.

---

## Fonctionnalités

- Interface graphique Pygame + pygame_gui en plusieurs écrans   
- Choix du mode de jeu :
  - Joueur vs Joueur  
  - Joueur vs IA   
- Gestion du tour de chaque joueur  
- Affichage dynamique de la grille et des symboles X et O
- Détection en cas de victoire ou de match nul
- Affichage d’une image en cas de victoire

## Prérequis

### Installation de Python 3  
- **Windows :** [Télécharger Python 3](https://www.python.org/downloads/windows/)  
- **macOS :** [Télécharger Python 3 pour macOS](https://www.python.org/downloads/macos/)  
- **Linux (source) :** [Télécharger les sources Python](https://www.python.org/downloads/source/)  

### Dépendances Python
Installer les librairies nécessaires :

```bash
pip install pygame pygame_gui
```

### Installation du projet

1. **Cloner le projet**

```bash
git clone https://github.com/lorenzo-brissardnavarro/tic-tac-toe.git
cd tic-tac-toe
```

2. **Lancer le projet**
```bash
python3 main.py
```
